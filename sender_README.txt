#README zu sender.py

Mit diesem Programm können Dateien an einen Computer der das Programm receiver.py am Laufen hat
geschickt werden. Angegeben werden muss dabei die Datei, die verschickt werden soll,
und die IP an die die Datei verschickt werden soll.

Diese können hinter den Parametern -ip(für die IP) und -p(für den Pfad) angegeben werden.
Optional ist der Parameter -I der regelt, ob die intern hinterlegte IP genutzt werden soll.
